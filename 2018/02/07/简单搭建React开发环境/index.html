搭建 React 项目开发环境

2018-02-07

第一步: 初始化

  1. 新建文件目录并进入
  2. yarn init 初始化
  3. 下载 webpack yarn add webpack --dev

此时目录结构为 

┌ node_modules 
├ src
|   ├ index.js  
└ package.json

第二步 配置 webpack

创建 webpack 的配置文件 webpack.config.js
并添加以下代码

const path = requier("path");  

module.exports = {  
  entry: "./src/index.js",  // 入口文件路径
  output: {  // 出口
    path: path.join(__dirname, "dist"),   // 出口文件路径
    filename: "build.js"  // 出口文件名
  }  
}

第三步 编译 JSX

此时的文档还不支持 JSX 语法，所以我们需要通过 babel 来编译它。通过下载以下插件来达到我们的目的: 

yarn add @babel/preset-react @babel/preset-env @babel/core babel-loader@8.0.0-beta.0 --dev

下载好之后创建 .babelrc 文件，并写入以下内容: 

{
  "presets": [
     "@babel/present-env",  
     "@babel/present-react"
  ]
}

然后修改 webpack 的配置文件 webpack.config.js，在配置出口文件的下方添加以下内容: 

module: {
  loaders: [
    {
      test: /\.js$/,  // 正则判断是否为 js 文件
      loader: "babel-loader",   // 使用 babel-loader
      exclude: /node_modules/   // 除 node_modules 以外编译
    }
  ]
}

现在我们已经支持了 js 文件的编译打包，下面来配置下 有关 css 和图片的相关配置吧。

下载 style-loader, css-loader 用来支持 css 文件打包 

yarn add style-loader css-loader --dev

下载 url-loader 来支持图片的引入

yarn add url-loader --dev

然后在配置文件中的 loaders 中添加 

{
  test: /\.css$/,  // 正则判断是否为 css 文件
  loader: [
    "style-loader",
    "css-loader"
  ]   // 使用 style-loader 和 css-loader
},
{
  test: /\.png$|\.jpg$|\.jpeg$|\.svg/,
  loader: "url-loader"
}

最后我们只需要在 index.js 中引入就可以了 

import css from "./index.css"

此时我们已经基本配置好 React 的开发环境了。但是下面的内容会让我们的开发过程更加舒服。

让我们来下载 html-webpack-plugin 用以将打包之后的文件自动插入到需要的模板中

yarn add http-webpack-plugin --dev

下载后在 webpack.config.js 中引入 

const HTMLWebpackPlugin = require("http-webpack-plugin")

然后在 module 块下方添加: 

plugins: [
  new HTMLWebpackPlugin({
    template: "./index.html",  // 模板路径
    filename: "index.html",   // 文件名
    title: "首页",   // 标题
    inject: "body"   // 选择插入的位置 
  })
]

现在我们已经实现了半自动了，可是我们还想全自动。怎么办呢？

下载 webpack-dev-server 让它支持热更新 

yarn add webpack-dev-server --dev

此时我们直接运行可能会出现报错，不用担心，只需要修改下 package.json 文件，在设置协议的下方添加: 

"scripts": {
  "start": "webpack-dev-server"
}

现在我们离成功只有一步之遥了，最后下载一下 react 和 react-dom ，然后在我们的 index.js 文件中引入，就可以正常使用啦。

yarn add react react-dom

现在 npm start 来运行吧~
